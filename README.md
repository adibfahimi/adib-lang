# Adib Lang

### Examples:

```js
// Print "Hello, world!"
print("Hello, world!")

// Conditional statements
var age = 18
if (age >= 18) {
  print("You are eligible to vote.")
} else {
  print("You are not eligible to vote.")
}

// For loop
for (var i = 0; i < 10; i++) {
  print(i)
}

// While loop
var i = 0
while (i < 10) {
  print(i)
  i = i + 1
}

// Function definition
function add(a, b) {
  return a + b
}

print(add(1, 2)) // Function call

// Arrays
var arr = [1, 2, 3, 4, 5]
print(arr[0])
```

## Standard Functions:

- `print`: Outputs the value to the console.
